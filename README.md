# LionLink Frontend

LionLink 백엔드와 통신하는 React + Vite SPA. 자료 공유 미리보기 보드.

## 기술 스택

- React 18 + Vite 5
- React Router v6
- Axios

## 환경변수

`.env.example`를 복사해 `.env`를 만든다.

- `VITE_API_URL` — 백엔드 base URL

## 로컬 실행

```bash
npm install
cp .env.example .env
npm run dev
```

## 빌드

```bash
npm run build
```
